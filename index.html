osdev-jp - 日本のOS開発コミュニティ

osdev-jpはOS開発に有用な情報を収集し公開しています。

トップページ

注意

このウェブサイトはTakymが勝手に作成した物で、非公式です。

目的は低レイヤに関する情報を公開したり、コミュニティ内の情報を共有する事です。

本家サイトはhttps://osdev.jpです。フォーク元のGitリポジトリはhttps://github.com/osdev-jp/osdev-jp.github.ioです。

本家サイトやWikiから情報が遅れている場合は、GitHub 上の Issuesでお願します。

このコミュニティはosdev.orgとは無関係です。

目次

  • Wiki - OS開発に役立つ情報が集まっています。GitHubアカウントがあれば編集できます。
  • Core Wiki - osdev-jp のコアメンバが執筆した技術文書を掲載する場所です。OS 開発に役立つ記事ばかりを扱っています。
  • GitHub - サンプルコードなどが置いてあります。
  • Slack - OS開発に関する話をするosdev-jpの公式チャットです。
    • チャンネル一覧
  • コミュニティガイドライン - このコミュニティの規則です。
  • 自作OS イベント＆ニュース - 自作OSに関係する出来事/最新情報をまとめています。
  • アドベントカレンダー一覧 - 自作OSに関係するアドベントカレンダーをまとめています。
  • Easy CPU
