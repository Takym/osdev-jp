osdev-jp - 日本のOS開発コミュニティ

osdev-jpはOS開発に有用な情報を収集し公開しています。

Slack チャンネル一覧

このページは、公式Slack内で利用されているチャンネルの一覧です。

良く使われるチャンネル

#event
各種イベントに関係する事を告知します。
#general
新規メンバーの自己紹介及びコミュニティに関わる重要なお知らせを扱います。
#random
低レイヤに関する事なら何でも話してもいい雑談用のチャンネルです。
#初心者質問相談
初心者専用の質問・相談チャンネルです。難しい事は扱いません。

その他

#acpi
Advanced Configuration and Power Interface に関係がある事を議論・質問します。
#coreboot
d0iasm の GSoC で coreboot を Qemu/Aarch64 上で動かすプロジェクトに関係がある事を議論・質問します。
#distributed-os
分散型オペレーティングシステムに関係がある事を議論・質問します。
#libcxx
libcxx に関係がある事を議論・質問します。
#live代替
Cybozu Live の代替品の開発について議論します。
#osc
オープンソースカンファレンスに関する情報を告知します。
#osdevide
Takym が開発している OSDeveloper IDE の開発について議論します。
#px86
x86 向けのエミュレータについて議論・質問します。
#rust
プログラミング言語「Rust」に関係がある事を議論・質問します。
#site-renewal
本家サイトの発展について議論します。
#uefi
Unified Extensible Firmware Interface に関係がある事を議論・質問します。
#九州大学鳥人間システム
九州大学の鳥人間システムプロジェクトに関係がある事を議論・質問します。
#自作vmm
エミュレータやハイパーバイザ系の話題を扱うチャンネルです。
#進捗発表
自作OSの進捗状況を発表・共有します。
